Run Profiler

On this page the TPF Profiler can be configured and executed to assess the performance of a pre-selection of TPF servers. There are four different knowledge graphs (DBLP, DBpedia, GeoNames and Wiktionary) available on two different TPF servers. One server is hosted locally and the other is publically available via the Linked Data Fragments page. Furthermore, the sample size to be used can be selected. For perfomance reasons, the maximum sample size is 100 and only one profiling process can be run at a time.
